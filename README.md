## Como Rodar o Projeto

O algoritmo foi implementado para ser executado diretamente no Node.js a partir da linha de comando.

## Pré-requisitos

Você deve ter o **Node.js** instalado em sua máquina.

## Estrutura de Arquivos

O projeto consiste em dois arquivos principais:

| Arquivo | Algoritmo |
| :--- | :--- |
| `index.js` | Arquivo do algoritmo gerado |

## 1\. Executando a Solução 

**Comando:**

```bash
node index.js
```
